如何更换主题

1.先进入Hexo官网主题


这就是Hexo官网主题下面的部分都是供你选择的一些主题

2.如何克隆主题
首先你得找到你自己心仪的主题(这里我们就用Smackdown这个主题来演示)，然后点击图片下面的那个蓝色的字体


然后进入这个页面


往下滑，找到这两个语句(框起来的第二个有些主题会没有，这时就得根据他在本地文件里面的名字来进行配置)


直接在终端中输入他给你的克隆命令(如果是我这样就说明克隆完成了)

3.接下来就得在本地的配置文件中进行文件的配置了
首先进入你博客根目录下的_config.yml文件


然后找到theme节点，将节点后面的配置换成之前我们克隆好的主题名字(“：”冒号之后空格不可少)


回到GIT BASH HERE中启动服务

$ hexo s

在浏览器中输入localhost:4000查看。(如果你的端口号更改过那么就用你更改过之后的端口)

在本地查看无误之后，输入生成和发布命令,就可已将新主题发布到自己的博客网站上了

$ hexo c

$ hexo g

$ hexo d

第一次执行hexo d命令的时候可能会要你登录验证你的Github，输入你Github用户名和密码即可。
然后在浏览器输入（你的用户名.github.io）就可以访问你最新的博客了。
